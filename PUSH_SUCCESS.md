## Key Features

### ✅ Dual-Track Methodology
- Architecture Track: OIR Framework
- Debugging Track: Evidence Ladder

### ✅ Decision Cost Filter
- Prevents over-analysis (C0: 5 min, C1: 30 min)
- Matches reasoning depth to problem cost

### ✅ Evidence-First Debugging
- P0: Stack trace (★★★★★)
- P1: Failing line (★★★★☆)
- P2: Variable state (★★★☆☆)
- P4: Root cause theory (★☆☆☆☆)

**Invariant**: No ROOT CAUSE until P0-P2 obtained.

### ✅ Calibration Tracking
- Target: >90% accuracy
- Weekly review process
- Drift detection built-in
